The mood of people in the State of Arizona by county over the last 24 hours.

Frequenty Asked Questions

How do you determine the mood in each county?

A sentiment index is computed on each tweet that originated in Arizona. This information is then aggregated at each county and displayed in the map.

How do you compute the sentiment index?

I am currently using the sentiment analyzer from the TextBlob API.

I see NaN when I hover over some counties. What does that mean?

When the application is unable to find a tweet in a certain geography it will print NaN as it cannot compute any information for that grographic area.

What tools/platforms did you use to build this mood map?

Django, Twython, Postgres and D3.js.

Do you have a REST API to access the mood information?

Not at the moment.

Why did you build this map?

Primarily to show how public information can be used to build a data application.

Will you be willing to share the code used to generate the map?

Absolutely. The code is available on github.
